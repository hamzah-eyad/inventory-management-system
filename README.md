# Inventory Management System

A console-based inventory management system built in C++ using Object-Oriented Programming (OOP) and Linked Lists.

## Features
- Add, remove, update, and display products
- Search for specific products
- Full CRUD operations
- Designed for small to medium-sized businesses

## Technologies Used
- Language: C++
- Concepts: Object-Oriented Programming (OOP), Linked Lists, Data Structures

## How to Run
1. Open the project in any C++ IDE (e.g., VS Code)
2. Compile `Project.cpp`
3. Run the executable
